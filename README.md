# react-native-radio-player

Play radio streams in react-native

## Installation

```sh
npm install react-native-radio-player
```

## Usage

```js
import RadioPlayer from "react-native-radio-player";

// ...

const result = await RadioPlayer.multiply(3, 7);
```
